# Collector-Control

This is a script for starting, stoping, and checking the status of a Contoller without using root.

## Usage

```console
./collctl.sh [start|stop]
```

## Examples
To start the Collector
```console
./collctl.sh start
```
To stop the Collector
```console
./collctl.sh stop
```
To check the status of the Collector
```console
./collctl.sh
```
